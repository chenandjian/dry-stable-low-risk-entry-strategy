# 量干 + 价稳评分规则

> 对应代码：`analyzer/volume_dry.py` / `analyzer/price_stable.py`
> 决策入口：`analyzer/decision.py`

---

## 一、量干评分（Volume Dry Score）— 0~10 分

判断卖压是否枯竭。成交量持续萎缩 = 卖方力量耗尽 = 买入时机临近。

### 评分维度

| 维度 | 满分 | 计算方法 | 阈值 |
|------|:--:|------|------|
| **A** 近期量vs均量 | 2 | V5（近5日均量）/ MA20 | ≤0.80: +2 / ≤0.90: +1 / >0.90: +0 |
| **B** 近期量vs长期 | 2 | V5 / MA50 | ≤0.70: +2 / ≤0.85: +1 / >0.85: +0 |
| **C** 量递减序列 | 2 | V1(15-11天前) → V2(10-6天前) → V3(近5天) | 严格递减: +2 / V2>V3: +1 / 不递减: +0 |
| **D** 下跌无量 | 2 | 近10天跌的日均量 / MA20 | 无跌日: +2 / ≤0.90: +2 / ≤1.05: +1 / >1.05: +0 |
| **E** 极端低量 | 2 | 近5天最小量 / MA50 | ≤0.50: +2 / ≤0.65: +1 / >0.65: +0 |

### 封顶规则

| 触发条件 | 效果 |
|----------|------|
| 近3天内有放量（≥1.5×MA20）且跌幅≥3% 的大阴线 | 总分最高 **5 分** |

### 判定

| 分数 | 判定 |
|:----:|------|
| ≥ 7 | 可低吸 |
| ≥ 6 | 观察 |
| < 6 | 不建议买入 |

## 二、价稳评分（Price Stability Score）— 0~10 分

判断价格是否停止下跌，进入窄幅稳定区间。波动收窄 = 方向选择前夜。

### 评分维度

| 维度 | 满分 | 计算方法 | 阈值 |
|------|:--:|------|------|
| **A** 振幅收缩 | 2 | Range5（近5天均振幅）/ Range20（近20天） | ≤0.60: +2 / ≤0.80: +1 / >0.80: +0 |
| **B** 无新低 | 2 | 近10天最低 vs 前段(20-30天前)最低 | Low10 > PrevLow20: +2 / ≥×0.98: +1 / <: +0 |
| **C** 低点上移 | 2 | 最近两个局部低点比较 | LocalLow2 > LocalLow1: +2 / ≥×0.98: +1 / <: +0 |
| **D** 站上MA20 | 2 | 近5天收盘是否在MA20之上 | 全部站上: +2 / 1次跌破后恢复: +1 / ≥2次跌破: +0 |
| **E** ATR收缩 | 2 | ATR5 / ATR20 | ≤0.75: +2 / ≤0.90: +1 / >0.90: +0 |

### 封顶规则

| 触发条件 | 效果 |
|----------|------|
| 近5天出现 20 日新低 | 总分最高 **5 分** |
| 连续下跌且跌幅放大（加速下跌） | 总分最高 **5 分** |

### 判定

| 分数 | 判定 |
|:----:|------|
| ≥ 7 | 可低吸 |
| ≥ 6 | 观察 |
| < 6 | 不建议买入 |

## 三、决策流程

```
                ┌─────────────────┐
                │  形态评分 ≥ 8   │──否──→ 不建议买入（形态不成熟）
                └────────┬────────┘
                         │是
                ┌────────▼────────┐
                │  量干评分 ≥ 6   │──否──→ 不建议买入（量能未干）
                └────────┬────────┘
                         │是
                ┌────────▼────────┐
                │  价稳评分 ≥ 6   │──否──→ 不建议买入（价格未稳）
                └────────┬────────┘
                         │是
                ┌────────▼────────┐
                │ 止损空间 ≤ 阈值 │──否──→ 不建议买入（止损空间超标）
                └────────┬────────┘
                         │是
                ┌────────▼────────┐
                │  RR1 ≥ 阈值     │──否──→ 不建议买入（盈亏比不足）
                └────────┬────────┘
                         │是
                ┌────────▼────────┐
                │ 量干≥7 价稳≥7   │
                │ 形态≥13 RR1达标 │──是──→ 可低吸
                │ 价格在入场区内   │
                └────────┬────────┘
                         │否（但形态≥13 + 价量≥6 + 未追涨）
                         │
                         ▼
                      突破确认
```

## 四、可配置阈值

`config.yaml` → `decision` 段：

```yaml
decision:
  min_pattern_score: 8           # 形态最低分（硬拒绝）
  min_volume_dry_score: 6        # 量干最低分（硬拒绝）
  min_price_stable_score: 6      # 价稳最低分（硬拒绝）
  max_risk_percent: 10           # 止损空间上限 %
  min_rr1: 2.0                   # 盈亏比下限
  chase_threshold_pct: 5         # 追涨阈值 %
  low_buy_min_pattern_score: 13  # 可低吸-形态门槛
  low_buy_min_volume_dry: 7      # 可低吸-量干门槛
  low_buy_min_price_stable: 7    # 可低吸-价稳门槛
  low_buy_max_risk_percent: 6    # 可低吸-止损上限 %
```

## 五、常见拒因与对策

| 拒因 | 根因 | 对策 |
|------|------|------|
| 量能未干（量干<6） | V5/MA50 偏高、无极端低量 | 等成交量进一步萎缩到 MA50 的 65% 以下 |
| 价格未稳（价稳<6） | 振幅没收窄、ATR 没降 | 等价格横盘整理 |
| 价格创新低（被封顶） | 近5天出20日新低 | 等新低后企稳 |
| 止损空间超标 | 入场区到止损距离太大 | 等股价回调到入场区内 |
